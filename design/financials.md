# Financials: Fees & Accounting

A clean, transparent interface for tracking collections, payments, and expenses.

## 1. Fees Directory (Admin View)
- **Status Tally Bar**: A row of small `p-3` metric boxes above the table (e.g., Total Receivable, Total Collected, Pending).
- **Payment Rows**:
  - Highlights the `dueAmount` in bold red if positive.
  - Features an action for "Collect Payment" which opens a mini-modal.

## 2. Student Fee View (Receipt Style)
- **Layout**: Centered "Bill Card".
- **Design**: Minimalist invoice style.
- **Pay Online Button**: Heavy branding using the corporate Emerald color, paired with a secure lock icon.

## 3. Expense Ledger
- **Category Filter**: Pills for filtering between "Utility", "Salary", "Rent", etc.
- **Visuals**: Negative numbers (expenses) shown in slate-600, while incomes are in emerald-600.
